# Package Examples

- `local-only.ts` shows the minimal `@fathippo/local` cache and edge-metrics flow.
- `hosted-hybrid.ts` shows queue setup, sync worker wiring, and hosted confidence gating with `@fathippo/hosted`.
- `cognition-enabled.ts` shows `@fathippo/cognition` client/engine setup plus a benchmark gate.

Compile all examples with:

```bash
npm run check:examples
```
