# Code for Machines, Not Just Humans: Quantifying AI-Friendliness with Code Health Metrics

## Cite_Key
borg2026code

## Problem
Ensuring that AI coding agents can reliably edit code in codebases optimized for human comprehension.

## Method
Investigated the concept of 'AI-friendly code' via LLM-based refactoring on a dataset of 5,000 Python files from competitive programming.

## Data
5,000 Python files from competitive programming.

## Metrics
Association between CodeHealth (a quality metric for human comprehension) and semantic preservation after AI refactoring.

## Findings
Human-friendly code is more compatible with AI tooling. Organizations can use CodeHealth to guide AI interventions with lower risk.
