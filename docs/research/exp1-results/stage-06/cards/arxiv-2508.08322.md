# Context Engineering for Multi-Agent LLM Code Assistants Using Elicit, NotebookLM, ChatGPT, and Claude Code

## Cite_Key
haseeb2025context

## Problem
LLMs struggle with complex, multi-file projects due to context limitations and knowledge gaps.

## Method
Proposed a context engineering workflow combining an Intent Translator, semantic retrieval, document synthesis, and a multi-agent system for code generation and validation.

## Data
Qualitative results on a large Next.js codebase.

## Metrics
Higher single-shot success rates and better adherence to project context.

## Findings
The multi-agent system effectively plans, edits, and tests complex features with minimal human intervention, outperforming baseline single-agent approaches.

## Limitations
Not specified.
